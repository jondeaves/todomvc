# Todo MVC

A collection of projects for a Todo MVC, sort of like a mono-repo. The aim of this project is to implement various parts of a Todo application in various languages.

## The plan

### UI

- [ ] React
- [ ] Angular
- [ ] Flutter (iOS, Android and Web)
- [ ] Android
- [ ] iOS

### Back-end

- [x] NestJS
- [ ] Dotnet core
- [ ] Rust
- [ ] Ruby-on-rails

### Database

- [x] SQlite
- [ ] Postgres
- [ ] MongoDB
